Conditions can be used to define if an element will be hidden. By default, every element is shown.

You can use the following tokens :
  • V as a value. Can be :
    • Variable (use the name you defined at the beginning of the scenario !)
    • Integer
  • COMP as Comparison. Can be :
    • == as equality test
    • != as inequality test
  • LOGIC as Logical. Can be :
    • && as comparison AND comparison
    • || as comparison OR comparison
You can use the following syntax :
  • V COMP V
  • V COMP V LOGIC V COMP V
Logical operators priority are respected : all && are evaluated before all || (as x would be evaluated before +)

Example : Variable1 == 1 || Variable1 == Variable2
